1. **Clone the repository**
   ```bash
   git clone https://github.com/Coder-Ankit-MCA/AI-ChatBot.git
   ```

   ```bash
   pip install -r requirements.txt
   ```

   ```python
   client = OpenAI(api_key="Your_API_KEY")

```
